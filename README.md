# APIGateway

A production-ready API Gateway built from scratch in Go, featuring request routing, JWT authentication, rate limiting, circuit breaker, and centralized observability — without relying on existing gateway frameworks.

## Stack

- **Go** — core language
- **Docker** — containerization
- **Redis** — rate limiting and caching
- **PostgreSQL** — persistent storage
- **OpenTelemetry + Grafana** — observability (metrics, traces, logs)
- **GitHub Actions** — CI/CD
- **Testify + Testcontainers** — testing
